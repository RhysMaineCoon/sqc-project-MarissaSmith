# My Project: A Text Analysis Tool

A tool that extracts and analyzes textual data from Project Gutenberg books to provide insights.

## Data source

- [Pride and Prejudice](https://www.gutenberg.org/ebooks/1342) by Jane Austen

## Items of interest

- Chapters
- Character dialogues
- Themes and motifs
